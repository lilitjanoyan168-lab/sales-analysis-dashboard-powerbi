# Sales Analysis Dashboard | Power BI

## Project Overview

This project analyzes sales performance across multiple countries and product brands using Power BI.

The dashboard was designed to monitor key business metrics and identify trends in revenue, profitability, transactions, and product returns.

## Tools Used

* Power BI
* DAX
* Power Query
* Data Modeling
* Excel

## Key Metrics

* Revenue
* Profit
* Profit Margin
* Transaction Volume
* Return Rate

## Key Insight

One of the most interesting findings was that brands with the highest transaction volumes were not always the most profitable. Higher return rates and lower profit margins reduced the overall value generated by some top-selling brands.

This highlighted the importance of evaluating profitability and return rates alongside sales volume when measuring business performance.
